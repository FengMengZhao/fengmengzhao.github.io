### 指导思想

> 1. 盖房子，先谈骨架，再谈修饰；画山水，先画结构，再谈润色
2. DO Forward! Think Forward!
3. 模仿要背着敲代码，而不是照着抄，那样就成了打字员
4. 抓大放小，要事为先
5. 锻炼产能，及时产出
6. 掌握各种知识的细节不一定就是高手
7. 从小到大养成的遇见细节就抠的性格，需要重新审视一下

### 如何成为一个优秀的开发者

> 1. 深入理解面向对象的核心理念
2. 掌握Java core API
3. 不停的编码(coding)
4. 订阅技术论坛和博客
6. 读开源框架的源代码
7. 了解技术发展的趋势
8. 积累属于自己的代码库
10. 写技术博客
